# Dashboard Icons

Aquí es donde debes poner tus iconos personalizados.

## Archivos necesarios:

1. **medication-icon.png** (o .svg, .jpg)
   - Icono para medicaciones
   - Tamaño recomendado: 40x40 px o más
   - Se mostrará en la sección "Medicaciones de hoy"

2. **appointment-icon.png** (o .svg, .jpg)
   - Icono para citas/appointments
   - Tamaño recomendado: 40x40 px o más
   - Se mostrará en la sección "Citas próximas"

3. **message-icon.png** (o .svg, .jpg)
   - Icono para mensajes sin leer
   - Tamaño recomendado: 40x40 px o más
   - Se mostrará en la sección "Mensajes sin leer"

## Instrucciones:

1. Reemplaza cada archivo con tu propia imagen
2. Usa los mismos nombres exactos (medication-icon, appointment-icon, message-icon)
3. Puedes usar PNG, SVG, JPG, etc.
4. Los iconos deben tener fondo transparente (recomendado)

¡Listo! Una vez que añadas tus imágenes, aparecerán automáticamente en el dashboard.
